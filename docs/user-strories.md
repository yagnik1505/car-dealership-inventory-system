# User Stories

## Authentication

### US-01: User Registration

As a new customer,
I want to register using my name, email, and password,
so that I can access the system securely.

---

### US-02: User Login

As a registered user,
I want to log in using my email and password,
so that I can access protected features.

---

## Vehicle Browsing

### US-03: View Vehicles

As a customer,
I want to view all available vehicles,
so that I can browse the dealership inventory.

---

### US-04: Search Vehicles

As a customer,
I want to search vehicles by make, model, category, or price,
so that I can quickly find vehicles matching my preferences.

---

## Vehicle Purchase

### US-05: Purchase Vehicle

As a customer,
I want to purchase an available vehicle,
so that the inventory is updated automatically.

---

## Inventory Management

### US-06: Add Vehicle

As an administrator,
I want to add new vehicles,
so that the inventory remains up to date.

---

### US-07: Update Vehicle

As an administrator,
I want to update vehicle information,
so that incorrect or outdated information can be corrected.

---

### US-08: Delete Vehicle

As an administrator,
I want to remove vehicles,
so that unavailable inventory is not displayed.

---

### US-09: Restock Vehicle

As an administrator,
I want to increase vehicle stock,
so that sold-out vehicles can become available again.
